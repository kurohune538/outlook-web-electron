# Outlook Web

Only for Mac

## To Use
you can run the latest App from `OutlookWeb-darwin-x64` > `OutlookWeb.app`.

### Build from source
#### Mac

```bash
$ electron-packager . OutlookWeb --platform=darwin --version=1.8.4 --overwrite --icon=./images/icon.icns
```

you need `electron-packager`
